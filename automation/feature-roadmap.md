# FLAI-TavernAI 功能路线图

参考 AI风月 (aisearches.xyz) 的核心功能，列出可借鉴的改进方向。
仅参考功能逻辑，不参考配色布局。

---

## 🔥 高优先级（核心体验）

### 1. 世界书系统 (World Book)
- **现状：** 无
- **目标：** 为角色卡附加世界观设定，支持触发词自动注入上下文
- **功能：**
  - 世界书 CRUD（条目名、触发键、内容、插入位置）
  - 与角色卡关联
  - 按触发词自动匹配注入
  - 支持递归扫描（条目间引用）
- **参考：** AI风月的世界书系统支持复杂嵌套引用

### 2. 角色卡标签与分类
- **现状：** 角色卡只有基础字段
- **目标：** 支持标签系统，方便发现和筛选
- **功能：**
  - 角色卡添加标签（如：纯爱、角色扮演、大世界、后宫等）
  - 按标签筛选
  - 标签云 / 热门标签
  - 分类浏览

### 3. 会话存档/读档 (Save/Load)
- **现状：** 无
- **目标：** 保存对话进度，支持多存档
- **功能：**
  - 保存当前对话为存档（含消息历史、世界书状态、变量）
  - 多存档位（存档1、存档2...）
  - 读档恢复
  - 存档预览（最后一条消息、时间戳）

### 4. 对话模板/预设 (Presets)
- **现状：** 无
- **目标：** 预设系统提示词和参数组合
- **功能：**
  - 系统提示词模板
  - 温度/max_tokens 等参数预设
  - 预设导入/导出
  - 与角色卡关联

---

## ⭐ 中优先级（体验提升）

### 5. 热门存档 / 社区分享
- **现状：** 无
- **目标：** 用户可分享角色卡和存档
- **功能：**
  - 导出角色卡为 JSON 文件
  - 导入角色卡
  - 未来：社区广场（可选）

### 6. Mod 系统
- **现状：** 无
- **目标：** 支持用户自定义扩展
- **功能：**
  - Mod 目录结构（提示词注入、文风增强、破甲等）
  - Mod 启用/禁用
  - Mod 与角色卡解耦
  - 社区 Mod 分享

### 7. 正则规则增强
- **现状：** 已有基础正则替换
- **目标：** 增强为完整的文风控制系统
- **功能：**
  - 正则规则分组（全局 / 角色专属）
  - 规则优先级排序
  - 启用/禁用单条规则
  - 批量导入/导出

### 8. 自定义状态栏 / UI 注入
- **现状：** 无
- **目标：** 在对话中嵌入自定义 UI 元素
- **功能：**
  - 状态栏模板（HP/MP/好感度等变量）
  - 变量自动更新（从 AI 回复中提取）
  - CSS 自定义注入
  - 动态 UI 渲染

---

## 💡 低优先级（进阶功能）

### 9. NPC 独立代理引擎
- **现状：** 无
- **目标：** NPC 作为独立 agent 自主行动
- **功能：**
  - NPC 独立记忆
  - NPC 自主行为决策
  - 多 NPC 同时活跃

### 10. 经济系统模拟
- **现状：** 无
- **目标：** 虚拟经济流动
- **功能：**
  - 货币系统
  - 交易记录
  - 经济事件触发

### 11. 天赋 Roll 系统
- **现状：** 无
- **目标：** 开局随机生成角色属性
- **功能：**
  - 天赋池定义
  - 随机 Roll 点
  - 属性影响对话

### 12. CG 立绘 / 图片系统
- **现状：** 只有头像上传
- **目标：** 角色多表情立绘
- **功能：**
  - 多张 CG 上传
  - 按场景切换立绘
  - 图片预加载

---

## 技术债 / 基础设施

### 13. 模型系数管理
- **现状：** 支持 OpenAI 兼容 API
- **目标：** 为不同模型配置消耗系数
- **功能：**
  - 模型系数配置（如 claude3.7 系数 0.042）
  - 消耗预估
  - 余额显示

### 14. 记忆管理增强
- **现状：** 基础对话历史
- **目标：** 可调节记忆深度
- **功能：**
  - 记忆值设置（1-5）
  - 自动摘要压缩
  - 长期记忆区 + 临时记忆区

### 15. 评论/评分系统
- **现状：** 只有点赞
- **目标：** 完整评论和评分
- **功能：**
  - 10 分制评分
  - 文字评论
  - 按时间/热度排序
  - 评论回复
