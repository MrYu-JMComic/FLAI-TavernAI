## 任务标题

对话预设管理页面（PresetView）

## 详细描述

FLAI-TavernAI 的"对话模板/预设"后端 CRUD 已完整实现（`backend/src/modules/presets.js` + `backend/src/routes/presets.js`），前端 API 函数已就绪（`frontend/src/api.js`），但**缺少前端视图和路由注册**，用户无法通过 UI 管理预设。

需创建 `frontend/src/views/PresetView.vue` 并完成路由集成，包含：

1. **预设列表**：展示所有预设，标记默认预设，显示创建时间
2. **新建预设**：表单（名称、系统提示词、temperature、max_tokens、top_p、frequency_penalty、presence_penalty）
3. **编辑预设**：点击进入编辑模式
4. **删除预设**：确认后删除
5. **设为默认**：一键设为默认预设（后端已处理互斥）
6. **路由集成**：App.vue 注册 `#/presets`，BaseLayout 侧边栏添加导航入口
