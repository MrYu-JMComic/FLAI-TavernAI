## 2. 任务规格

### 2.1 现状诊断

通过代码审查，当前前端错误处理存在以下问题：

| 问题 | 位置 | 影响 |
|------|------|------|
| **静默吞错** | `CharacterFormView.vue` 第 109/120/144/150/161 行 `catch {}` | 用户操作失败无任何反馈 |
| **错误消息不一致** | 部分用 `notify.error(err.message)`，部分硬编码中文，部分无提示 | 用户无法判断是否出错 |
| **网络错误缺乏重试** | 所有 API 调用失败后只能手动刷新 | 断网/后端重启后体验差 |
| **流式对话错误** | `ChatView.vue` 中 streaming 失败的错误显示为 toast，无持久化错误状态 | 对话中断后用户可能错过错误信息 |
| **401 未统一处理** | API 返回 401 时部分页面跳登录，部分仅 toast 提示 | 认证过期行为不一致 |
| **加载/空/错误三态** | 各视图自行管理，无统一模式 | 代码重复，样式不统一 |
| **SSE 流错误信息** | `streamSSE` 中 `throwOnError` 仅在 assistant draft 路径启用 | 对话流式错误缺少结构化信息 |

### 2.2 具体工作

#### A. 统一 API 错误处理层 (`api.js`)

1. **401 全局拦截**：在 `apiRequest` 中增加 401 检测，触发全局登出+跳转
2. **结构化错误对象**：`throwApiError` 增加 `code` 字段（`NETWORK`/`HTTP_4xx`/`HTTP_5xx`/`CSRF`/`AUTH`/`STREAM`），便于消费方按类型处理
3. **重试标记**：错误对象增加 `retryable` 布尔字段，标记 502/503/网络错误可重试
4. **SSE 错误标准化**：`streamSSE` 统一 `throwOnError=true`，`error` 事件始终抛出结构化错误

#### B. 视图层错误处理规范化

5. **消灭静默 `catch {}`**：`CharacterFormView.vue` 中 6 处空 catch 块改为 `notify.error` 或条件静默（如网络探测类调用）
6. **ChatView 流式错误持久化**：streaming 失败时在消息区域显示可重试的错误条，而非仅 toast
7. **统一重试模式**：关键操作（加载对话、发送消息）失败后提供"重试"按钮
8. **HomeView 加载失败重试**：角色列表加载失败时显示错误状态+重试按钮（已有 `loadError` 但无重试）

#### C. 全局错误通知优化

9. **Toast 消息分级**：`notify.error` 区分网络错误（提示检查连接）和业务错误（直接显示后端消息）
10. **401 跳转统一**：所有 401 响应统一跳转登录页，携带 `?redirect=` 参数

### 2.3 不做的事

- 不重构视图组件结构（空/加载/错误三态统一是独立任务）
- 不引入新的 UI 组件库
- 不改动后端 API 响应格式
- 不添加 accessibility 检查（独立任务）

## 4. 验收标准

| # | 场景 | 预期 |
|---|------|------|
| A1 | API 返回 401 | 自动跳转登录页，URL 含 `?redirect=原页面` |
| A2 | API 返回 502 | 错误对象 `code='HTTP_5xx'`, `retryable=true` |
| A3 | 网络断开（fetch 失败） | 错误对象 `code='NETWORK'`, `retryable=true`，提示"无法连接后端" |
| A4 | API 返回 400 业务错误 | 错误对象 `code='HTTP_4xx'`, `retryable=false`，直接显示后端消息 |
| B1 | CharacterFormView 保存时后端返回 500 | 用户看到具体错误 toast（非静默） |
| B2 | CharacterFormView 加载头像失败（网络探测） | 静默忽略，不影响页面（`catch` 中标注 `// 非关键操作`） |
| B3 | ChatView 发送消息后流式返回 500 | 消息区域显示错误条 + "重试"按钮 |
| B4 | ChatView 点击"重试" | 重新发送最后一条用户消息 |
| B5 | HomeView 首页加载角色列表网络失败 | 显示错误状态 + "重试"按钮 |
| B6 | HomeView 点击"重试" | 重新加载角色列表 |
| B7 | WorldBookView 加载世界书列表 502 | 显示错误 + "重试"按钮 |
| C1 | 任何 API 502 错误 | toast 显示"后端连接中断，请稍后重试" |
| C2 | 任何 API 网络错误 | toast 显示"无法连接后端服务，请确认 3001 后端窗口正在运行" |
