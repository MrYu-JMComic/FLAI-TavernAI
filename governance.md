# FLAI-TavernAI 三省六部制

基于 [Edict](https://github.com/cft0808/edict) 的完整三省六部多 Agent 协作系统。

---

## 架构

```
你 (皇上) → 太子 (分拣) → 中书省 (规划) → 门下省 (审议) → 尚书省 (派发) → 六部 (执行) → 回奏
```

## 分权制衡

- **中书省** 只能规划，不能执行
- **门下省** 只能审核，可以封驳（打回）
- **尚书省** 只能派发和协调，不能自己干
- **六部** 只能执行，不能越权决策
- 任何变更必须经过门下省审核
